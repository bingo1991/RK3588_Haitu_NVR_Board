# RK3588_Haitu_NVR_Board

[HT-RK3588](http://www.haitutech.com/proinfo/74.html)是常州海图信息科技股份有限公司推出的一款RK3588产品
- CPU: Rk3588
- Mem: 8GB LPDDR4
- EMMC: 32GB
- Power: 12V DC
- Network: 2 x 1GbE(RTL8211)
- Display Out: 2 x HDMI 2.1
- Display In: 2 x HDMI 2.0, one with LT6911UXC
- SATA Power: 1 x 4Pin
- SATA: 2 x SATA 3.0
- USB 3.0 Type-A: 1
- USB 2.0 Type-A: 2
- USB Type-C OTG: 1
- Button: MaskRom, Reset, Recovery, PMIC_Pwr_On
